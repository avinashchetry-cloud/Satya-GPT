#SatyaGPT - AI Powered Fake News Detection

## Overview

SatyaGPT is an AI-powered fake news detection platform developed during a hackathon. The application helps users verify the authenticity of news articles by analyzing their content using artificial intelligence.

The project was selected for the **Final Round** of the hackathon.

## Features

* AI-powered fake news detection
* Fast backend API for news analysis
* Clean and scalable backend architecture
* Easy integration with AI models
* REST API support for client applications
